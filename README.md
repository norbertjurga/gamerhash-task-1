## GamerHash - Task #1

Installation
```
cp .env.example .env
composer install
./vendor/bin/sail up 
./vendor/bin/sail artisan migrate
```

Run tests
```
./vendor/bin/sail artisan test
```

Seed the database
```
./vendor/bin/sail artisan db:seed
```
